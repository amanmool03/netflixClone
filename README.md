#Netflix clone React, firebase, Tailwind

This is a repository for Netflix clone. 

Key Features:
- 📡 Streaming Actual movies provided by third party movie streaming platform 
- 🔐 Authentication 
- 📸 Lazy image loading
- 🚦 Live statuses 
- 👥 Recommendation system.
